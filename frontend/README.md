# 📚 Biblioteca Aurea

Sistema completo di gestione biblioteca con interfaccia elegante, ruoli utente e pannello amministratore.

---

## 🚀 Funzionalità principali

### 👤 Utente
- Registrazione e login
- Visualizzazione catalogo libri
- Noleggio libri
- Restituzione libri
- Visualizzazione storico noleggi
- Sistema punti fedeltà

### 🛠️ Admin
- Aggiunta nuovi libri
- Visualizzazione noleggi attivi
- Dashboard con statistiche
- Controllo completo della biblioteca

---

## 🎨 UI / UX

- Design minimal elegante (stile Treccani)
- Dark mode 🌙
- Multilingua:
  - 🇮🇹 Italiano
  - 🇬🇧 Inglese
  - 🇫🇷 Francese
- Responsive (mobile + desktop)

---

## 🧱 Tech Stack

### Frontend
- React + TypeScript
- Material UI (MUI)
- React Router
- i18next (multilingua)

### Backend
- Node.js + Express
- Prisma ORM
- MySQL
- Express Session (autenticazione)
